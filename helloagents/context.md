# 项目上下文

## 1. 基本信息

```yaml
名称: 在线粘贴板 (PasteBin)
描述: 支持管理员与游客的安全在线粘贴板，用于跨设备保存与分享文本/代码/Markdown
类型: Web应用
状态: 开发中
```

## 2. 技术上下文

```yaml
语言: TypeScript
框架: Next.js 16 (App Router)
包管理器: pnpm
构建工具: Next.js (内置)
```

## 3. 项目概述

### 核心功能
- 管理员与游客双角色
- 粘贴创建/编辑/删除/查看（文本/代码/Markdown）
- 附件上传与下载（管理员）
- 分享链接（公开/密码/一次性/过期）
- 管理后台（系统配置/审计日志/批量清理）

### 项目边界
```yaml
范围内:
  - 正文加密存储与安全访问
  - 游客临时使用与管理密钥
  - 分享链接的可组合策略
范围外:
  - 主题定制/全文搜索/多管理员/2FA
```
